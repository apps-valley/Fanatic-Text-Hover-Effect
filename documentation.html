About

Text Hover Effect is a Sass mixin library that consists of effects for typography that are triggered on hover. Effects are made with pseudoelements ::after and ::before and CSS3 transitions. There are over 50 effects you can choose from: overline, doubleOverline, underline, doubleUnderline, oneSide, bothSide, linethrough, doubleLinethrough, special, fills, fillCorners, cornersClose, cornersExpand.

See demo for more

Getting Started

Download or clone repo from Github and include effect.scss file. In order effects to work you need to have partials directory in your project directory on the same level as effect.scss file. Effects.scss imports all of the mixins. If you need only several effects, delete unnecessary imports. All effects must have _base.scss imported because it has transition-opt and line-opt mixins mandatory for all effects.

           
             
              /* style.scss */

               @import "effects";
             
             
           
           
             
              /* structure */

              |--partials/
              |--effects.scss
              |--style.scss

How To Use

After you have all the files downloaded, you are ready to use effects. Include transition-opt to an element you wish to apply effect for. Then add transition. For example:

               
                 
                   .element { 
                       @include transition-opt;
                       @include underline-right;
                 }
Result

Be careful with using CSS #id selector because it will overwrite several styles. It is recommended to use classes instead.

Customize

Mixin transition-opt has several options. These are defaults

             
               
                 .element { 
                     @include transition-opt(
                     $font-size: 4em ,
                     $font-weight: 700,
                     $color: #2C3E50,
                     $hover-color: #FFF,
                     $border-after: 0%,
                     $border-before: 0%,
                     $display: inline,
                     $transition-color: #FFF,
                     $transition-speed: 1s,
                     $tranition-easing: ease-in-out,
                     $delay-after: 0s,
                     $delay-before: 0s,
                     $padding: 10px
                 )};

Change whichever property you want but be careful with changing display property. It is recommended to use display: inline because then width of transition will be equal to the width of an element. If you change to display: block, it will change the width of transition and then you must change width and margin of an element. For example:

             
               
                 .element { 
                     width: 100%;
                     @include transition-opt(
                     $display: block
                 )};
Result 
             
               
                 .element { 
                     width: 50%;
                     margin: 0 auto;
                     @include transition-opt(
                     $display: block
                 )};
Result

                       $transition-color: #FFF
                   )
                   @include fill-closing-horizontal;
